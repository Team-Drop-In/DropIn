## 🌲 Git
### Branch
- `main` : 서비스 운영 브랜치입니다.
- `dev` : 개발 환경 브랜치입니다. 개별적으로 작업했던 내용을 합치고 검토합니다.
- `feat/fe(or be)/...` : 프론트(백엔드) 세부 브랜치입니다.
### Commit & Pull-Request Message
| "feat: ~ " | 새로운 기능 추가 |
| :---: | :--- |
| "fix: ~ " | 버그 수정 |
| "design: ~ " | css와 같은 UI 수정 |
| "docs: ~ " | 문서 수정 (ex. README.md) |
| "remove: ~ " | 파일을 삭제만 한 경우 |
